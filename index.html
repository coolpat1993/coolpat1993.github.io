How to Play

A question will appear on screen. Once it appears, the timer starts.

There are three question formats:

Letters

Input only the first letter of your answer.

  • If the answer is a person's name, input the first letter of their first name.
  • If the answer begins with "The", for example "The Lion King", answer "L" not "T".

Multiple Choice

Listen to the multiple choice options and input A, B, C, D, E, or F on the keypad.

Numbers

Type in a number and press Enter.

The original smartphone pub quiz

How to Play

A question will appear on screen. Once it appears, the timer starts. Answer as quickly as possible to earn more points!

Letters

Input only the first letter of your answer.

  • If the answer is a person's name, use the first letter of their first name.
  • If the answer begins with “The”, e.g. “The Lion King”, answer “L” not “T”.

Multiple Choice

Select the correct answer from the options.

Numbers

Type in a number and press Enter.

Sequence

Put the options in the correct order by tapping them in sequence.
